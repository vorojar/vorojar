# Hi, I'm vorojar wu

I build small, useful, AI-native tools for real workflows.

I am based in Ningbo, China, working at 易哈佛. Recently I have been spending most of my maker time on local-first AI apps, agent workflows, document/OCR tooling, Markdown tools, and practical desktop utilities.

我现在更喜欢做一类东西：不是演示，不是概念图，而是能被每天打开、能节省时间、能替代昂贵或笨重软件的小工具。

## What I Build

- Local-first AI products: OCR, dictation, audiobook generation, subtitle automation.
- Agent tooling: long-running assistants, workflow orchestration, multi-channel bots.
- Lightweight native apps: Rust, Go, Swift, Tauri, WebView, and no-Electron experiments.
- Open-source utilities that turn messy daily work into one-click workflows.

## Current Direction

I am especially interested in:

- turning agents from "chat boxes" into reliable operators;
- making local AI useful on ordinary machines;
- replacing expensive SaaS or bloated desktop apps with focused open-source tools;
- building workflows where the model, CLI, browser, files, and user all cooperate cleanly.

## Principles

- Useful beats impressive.
- Local and private is worth fighting for.
- Small native tools can still feel polished.
- Good automation should save attention, not create more dashboards.
- If a tool is used every day, details matter.
